Política de Privacidade

Esta Política de Privacidade descreve como o aplicativo coleta, armazena e protege os dados dos usuários.

1. Informações Coletadas

Nosso jogo coleta apenas as seguintes informações dos usuários:

  • Nome do jogador
  • Pontuação

Esses dados são armazenados em um banco de dados externo apenas para manter o ranking do jogo.

2. Uso das Informações

As informações coletadas são utilizadas exclusivamente para:

  • Registrar a pontuação do jogador
  • Exibir o ranking dentro do jogo

3. Compartilhamento de Dados

Não compartilhamos os dados pessoais dos usuários com terceiros.

4. Exclusão de Conta e Dados

Se você desejar solicitar a exclusão da sua conta e todos os dados associados, clique no link abaixo ou envie um e-mail:

Solicitar exclusão de conta e dados

Nós processaremos sua solicitação em até 30 dias.

5. Segurança dos Dados

Tomamos medidas razoáveis para proteger os dados dos usuários contra acesso não autorizado, alteração ou divulgação. No entanto, nenhum método de transmissão ou armazenamento eletrônico é 100% seguro.

6. Alterações nesta Política

Podemos atualizar esta política de privacidade periodicamente. Recomendamos que os usuários revisem esta página regularmente.
